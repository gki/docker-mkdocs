codehilite is a plugin for syntax highlight.

## JavaScript with Language Hint
    ``` javascript
    console.log("Hello world!")
    ```

``` javascript
console.log("Hello world!")
```

    {
      "name": "Bob",
      "age": 20,

      "Hobby": ["Fishing", "Running", "BBQ"]

## Highlight lines
    ``` hl_lines="1 3"
    public class MyClass {
      public void main(String[] args) {
        System.out.println("hello world!")
      }
    }
    ```

``` hl_lines="1 3"
public class MyClass {
  public void main(String[] args) {
    System.out.println("hello world!")
  }
}
```
